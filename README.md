# Digidesign

Application de génération d'images selon la charte graphique Digiforma.

## Structure

```
Digidesign/
├── app/                    # Code source de l'application
├── assets/                 # Ressources graphiques
│   ├── logos/              # Logos Digiforma (SVG, PNG)
│   ├── photos/             # Photographies
│   ├── illustrations/      # Illustrations vectorielles
│   └── videos/             # Vidéos et motion
└── brand-guidelines.md     # Charte graphique complète
```

## Objectif

Digidesign génère automatiquement des visuels (posts, bannières, stories…) en respectant strictement la charte graphique Digiforma — couleurs, typographie, ton et composition.
